# cs
## 指令-路徑  <br /> 
1.絕對路徑（由根目錄寫起）  <br />
2.相對路徑（相對於目前的工作目錄的路徑）  
## 特殊的目錄  
* .  :此層目錄
* .. :上一層目錄
* -  :前一個工作目錄
* ～ :目前使用者所在的家目錄
## Linux指令
* cd 移動
* pwd 顯示當下位置
* ls 顯示當下目錄所有檔案
* cp 複製
* cat 查看檔案內容
* mkdir 建立一個新目錄
* mkdir-p 同時建立多個目錄
* rmdir 刪除一個空目錄
* rm 刪除檔案 
* mv 移動/重新命名
* nano 編輯文檔
* gzip FileName 壓縮檔案
* gunzip FileName.gz 解壓縮
* find 檔案搜尋
* more+20 testfile 從第20行開始顯示testfile文檔內容
* head-n 取前面幾行數字
* tail-n 取後面幾行數字
